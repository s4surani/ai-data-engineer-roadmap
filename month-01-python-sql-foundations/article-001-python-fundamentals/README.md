# Article 1: Python Fundamentals for Data Engineering

## What This Code Does
- Demonstrates Python variables and data types
- Reads CSV files using Pandas
- Performs basic data analysis

## How to Run
1. Activate virtual environment: `source venv/bin/activate`
2. Install dependencies: `pip install -r requirements.txt`
3. Run script: `python data_processor.py`

## Files
- `data_processor.py` - Main script
- `sample_sales.csv` - Sample data
- `requirements.txt` - Python dependencies
